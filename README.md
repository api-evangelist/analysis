# Analysis (analysis)
A curated index of services, tooling, and open source solutions for API analysis, analytics, linting, traffic inspection, and intelligence. This topic collection covers the full spectrum of API observability — from real-time traffic capture and replay (Wireshark, mitmproxy, Charles Proxy, GoReplay) to product analytics (Amplitude, Mixpanel, PostHog, Plausible), API monitoring and APM (Datadog, New Relic, Dynatrace, AppDynamics), API security analysis (42Crunch, Salt Security, Noname Security, Traceable), and linting and governance (Spectral, Redocly, Vacuum, Optic).
